# Dealer OS — DOC F&I Allocation

Upload `index.html` to the root of the GitHub Pages repository.

## Changes
- DOC F&I remains the single financial source of truth.
- Main F&I card now always shows DOC actual and DOC forecast.
- F&I log is used only to allocate the DOC total between New Retail and Used for analysis.
- New/Used allocation shows finance penetration, products per customer, actual £/unit, forecast £/unit and allocated forecast.
- The allocated New/Used amounts are not added back into profit and cannot double-count the DOC total.
